                         QUICK START (2 MINUTES)
================================================================================

STEP 1: Add Your OpenAI API Key
--------------------------------
1. Open the file named ".env" in this folder
2. Replace "your-openai-api-key-here" with your actual API key
3. Get a key from: https://platform.openai.com/api-keys

Your .env file should look like:
    OPENAI_API_KEY=sk-abc123xyz...

STEP 2: Run the Diagnostic
---------------------------
Open terminal/command prompt in this folder and run:

    python check_setup.py

This will verify everything is configured correctly.

STEP 3: Start the Server
-------------------------
Windows:
    start_server.bat

Mac/Linux:
    chmod +x start_server.sh
    ./start_server.sh

Wait for: "Uvicorn running on http://0.0.0.0:8000"

STEP 4: Open in Browser
------------------------
Go to: http://localhost:8000

Click "Chat with My AI Assistant" and try asking:
- "What is your current role at Arctic Wolf?"
- "Tell me about your leadership experience"

================================================================================
                           TROUBLESHOOTING
================================================================================

If something doesn't work:

1. Run: python check_setup.py
2. Check the terminal for error messages
3. See TROUBLESHOOTING.md for detailed solutions

Common issues:
- "API key not found" → Edit .env file with your key
- "Module not found" → Run: pip install -r requirements.txt
- "Cannot connect" → Make sure server is running
- "Model not found" → config.yaml already fixed to use gpt-4o-mini
